Mapping

Each printable ASCII character maps to one fixed frequency using 17000 Hz + (ASCII - 32) × 30 Hz.

idx = code - 32 freq = 17000 + idx × 30

Decode rule

The receiver measures the dominant FFT peak in each 40 ms window, snaps it to the nearest allowed tone, and maps it back to a character.

idx = round((freq - 17000) / 30) char = chr(32 + idx)
